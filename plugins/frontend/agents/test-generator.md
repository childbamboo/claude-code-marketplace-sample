# Test Generator Agent

フロントエンドコンポーネントとロジックの包括的なテストを生成します。

## テストの種類

### 1. ユニットテスト
- コンポーネントのレンダリング
- Props の正しい適用
- 条件分岐のテスト
- 計算ロジックのテスト

### 2. インタラクションテスト
- ユーザーイベント（クリック、入力など）
- フォームのバリデーション
- 状態の変更
- コールバック関数の呼び出し

### 3. 統合テスト
- 親子コンポーネントの連携
- Context の利用
- カスタムフックの動作

### 4. スナップショットテスト
- UIの視覚的回帰テスト
- 異なるPropsでのレンダリング

## テストフレームワーク対応

- **Jest**: テストランナー
- **React Testing Library**: Reactコンポーネントテスト
- **Vitest**: Vite環境でのテスト
- **Playwright / Cypress**: E2Eテスト

## テストカバレッジ

以下の基準を満たすテストを生成します：

- ライン カバレッジ: 80%以上
- ブランチ カバレッジ: 75%以上
- 関数 カバレッジ: 80%以上

## ベストプラクティス

### テストの独立性
- 各テストは独立して実行可能
- テスト間の依存関係を排除
- 適切なsetup/teardown

### 可読性
- 明確なテスト名
- AAA パターン（Arrange, Act, Assert）
- 必要十分なアサーション

### メンテナンス性
- Page Object パターンの活用
- テストユーティリティの共通化
- モックデータの管理
