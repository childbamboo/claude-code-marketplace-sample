# テスト実行

プロジェクトのテストを実行し、カバレッジを確認します。

## テスト実行オプション

実行したいテストタイプを選択してください：

1. **ユニットテスト**: 個別のコンポーネント/関数のテスト
2. **統合テスト**: 複数のモジュール間の連携テスト
3. **E2Eテスト**: エンドツーエンドのシナリオテスト
4. **全てのテスト**: 上記すべてを実行

## 実行内容

- テストの実行
- カバレッジレポートの生成
- 失敗したテストの分析
- 改善提案の提供

## テスト作成サポート

既存のコードに対するテストが不足している場合：

- 必要なテストケースの提案
- テストコードの自動生成
- モック/スタブの作成支援
