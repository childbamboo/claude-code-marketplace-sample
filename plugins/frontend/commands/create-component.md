# コンポーネント作成

新しいUIコンポーネントの作成をサポートします。

## 作成するコンポーネント情報

以下の情報を教えてください：

1. **コンポーネント名**: 例）UserProfile, ProductCard, NavigationBar
2. **コンポーネントタイプ**:
   - Presentational（表示のみ）
   - Container（ロジック含む）
   - Page（ページコンポーネント）
3. **使用するフレームワーク/ライブラリ**: React, Vue, Svelteなど
4. **必要な機能**: Props, State, イベントハンドラーなど

## 自動生成される内容

- コンポーネントファイル（.tsx, .jsx, .vueなど）
- スタイルファイル（該当する場合）
- テストファイル
- Storybook用のストーリーファイル（該当する場合）

## ベストプラクティス適用

- TypeScript型定義
- アクセシビリティ対応（ARIA属性など）
- レスポンシブデザイン考慮
- パフォーマンス最適化（React.memoなど）

作成したいコンポーネントの詳細を教えてください！
