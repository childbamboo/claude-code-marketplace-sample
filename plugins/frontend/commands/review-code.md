# コードレビュー（Web開発）

Web開発の観点でコードをレビューし、フィードバックを提供します。

## レビュー観点

### 1. フロントエンド品質
- コンポーネント設計
- 状態管理の適切性
- 再利用性とモジュール性

### 2. パフォーマンス
- レンダリング最適化
- バンドルサイズ
- 遅延ロード/コード分割
- 画像最適化

### 3. アクセシビリティ
- セマンティックHTML
- ARIA属性
- キーボードナビゲーション
- スクリーンリーダー対応

### 4. UX/UI
- レスポンシブデザイン
- ローディング状態
- エラーハンドリング
- ユーザーフィードバック

### 5. セキュリティ
- XSS対策
- CSRF対策
- 機密情報の取り扱い
- 依存関係の脆弱性

### 6. ベストプラクティス
- TypeScript型安全性
- Linter/Formatter設定
- テストカバレッジ
- ドキュメント

レビュー対象のファイルやコードを教えてください。
